- `git init`:
	  创建一个 git 仓库
- `git clone 网址`:
	- 克隆一个网上的项目
- `git add 文件名`：
	- 将文件添加到 git 仓库中，. 表示添加所有
- `git rm`：
	- 将文件从 git 仓库中删除
- `git commit`：
	- 提交修改的文件
- `git reset head~ --soft`：
	- 撤销上一次提交，但无法撤销第一次提交
- `git status`：
	- 查看文件状态
		- 红色：已修改但未暂存
		- 绿色：已暂存但未提交
- `git diff`：
	- 查看文件的具体修改
- `git log`：
	- 查看提交历史
- `git branch 分支名`:
	- 创建一个新的分支
- `git branch --list`：
	- 显示当前仓库的所有分支，绿色的为当前所在分支
- `git checkout 分支名`:
	- 切换到指定的分支
- `git merge 分支名`：
	- 将当前分支与指定分支合并
- `git fetch`：
	- 拉取远程仓库的更新，但不与本地分支合并
- `git pull`：
	- 拉起远程仓库的更新，并合并到本地分支
- `git stash`：
	- 用于需要切换到其他分支，但对当前分支的修改暂未完成时临时储存当前修改
- `git stash apply`：
	- 恢复使用 git stash 临时储存的修改
