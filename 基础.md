	单行注释：//
	多行注释：`/*   */`
	文档注释： `/** */`
关键字：
	package：
		后面跟包名，代表当前类定义在哪个包（文件夹）下
	public：
		相对复杂，暂且按下不表
	class：
		定义一个类，类是java中最基本的组成单元
```
		pubilc class 类名{
		}
```
	`public static void main`:
		表示java程序的主入口，当程序开始运行的时候，会从主入口开始向下执行
