运算：
	与Python不同，java的数据类型较多，不同数据类型没法直接进行运算，比如同属整型的int和byte就无法直接进行运算，如果要进行计算会将范围小的数据类型转为范围大的数据类型进行计算，java自动进行。
	在ASII表中有的字符进行计算时会转为对应数字，注意，特指字符而不是字符串
	字符串只有+运算，与任何数据进行运算时会进行拼接操作，如123+"你好" = “123你好”

		与python相同
	逻辑运算符：
		& 与
		 | 或

	短路逻辑运算符：
		逻辑运算符在java中执行效率较低，例如a = 1，判断a > 2 & a <10时，左边已经是false，但右边会继续进行判断，然后再对&进行判断，为了提高效率，有了短路逻辑运算符：
			&&：
				短路与，左边为false时，右边不再执行
			||：
				短路或，左边为true时，右边不再执行
	三元运算符：
		可以理解为简单的if语句，具体语法为：
			关系表达式 ?表达式1:表达式2;
			`a>b?a:b;`
		如果关系表达式a>b的结果是true，则返回表达式1a，否则返回表达式2b

switch语句：
	switch(变量){
		case 1:
			语句;
			break;
		case 2:
			语句;
			break
		default:
			语句;
			break;
	}
	当所有case都与变量不匹配时，就会执行default，default不是必须的
	break需要加上，不然会出现case穿透现象，例如变量为1，case1的语句中没有break,那么后续case中的语句也会依次执行。
	case值必须是确定的常量，switch语句不支持double之类的浮点数类型！！！！
	支持int，char，str，enum	
	java12+版本的新语法：
		switch(变量){
			case 1,2,3,4 -> 语句;
			case 5,6,7,8 -> 语句;
			default -> 语句;
		};
	这种语法不会触发case穿透，但仅在较高版本的java中可用
	只有语句是单行可以这样写，多行还是要加大括号
	yield关键字：
		在java13+中引入，作用是从switch中返回一个值
		String name = switch (month) {  
		    case 1, 2, 3 -> {  
		        yield "春季";  
			    }  
		    case 4, 5, 6 -> {  
		        yield "夏季";  
			    }  
		    case 7, 8, 9 -> {  
		        yield "秋季";  
			    }  
		    case 10, 11, 12 -> {  
		        yield "冬季";  
			    }  
		    default -> {  
		        yield "输入错误";  
			    }  
			 };
		yield与return的不同：
			yield: 只在 switch 表达式中使用，返回值给 switch 表达式
			return: 在方法中使用，返回值给方法调用者
		if与switch的使用场景：
			一般讲if可以完全替代switch,switch相较于if将其优点只有阅读性更好，性能更高，因此在有限个case时使用switch而不是if,可以提高运行效率

数组：
	数组的静态初始化：
		静态初始化指在创建的同时进行赋值，而不是创建空数组再向里面添加数据
		静态初始化的数组长度是固定的
		数组也存在数据类型，但向里面存数据时java会自动进行隐式转换
		完整静态初始化格式：
			数据类型 [ ] 数组名 = new 数据类型 [ ] {值1,值2} 
			int [] arr = new int [] {1,2,3}
		简化格式：
			数据类型 [ ] 数组名 =  {值1,值2} 
			int [] arr = {1,2,3}
	数组中元素的访问：
		java中数组索引从0开始
		获取：
			数组名[索引]
			arr[5]
		修改：
			数组名[索引] = 值
			arr[5] = 3
数组的遍历：
		for (i=0; i < arr.length; i++){
		}
		arr.length可以获取数组长度
动态初始化：
	动态初始化指在创建数组时只指定长度，有java自动赋初始值，整数为0,小数为0.0
	数组类型 [] 数组名 = new 数据类型 [数组长度]\
	int [] arr = new int [3];
