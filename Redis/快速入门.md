- 应用场景：
	- 缓存
	- 任务队列
	- 消息队列
	- 分布式锁
- 常用数据类型：
	- 字符串（String）
	- 哈希（hash）
	- 列表（list）
	  集合（set）
	- 有序集合（sorted set）
- 字符串常用操作命令（未加粗部分为具体值，例如 SET name xiaoming）：
	- **SET** key value                    ： 设定指定key的值
	- **GET** key                             :    获取指定key的值
	- **SETEX** key seconds value  :   设置指定key的值，并将key的过期时间设置为seconds秒
	- **SETNX** key value               ： 只有在key不存在时设置value的值
- 哈希常用操作命令：
	- 简要概述：哈希是一种键值对类型的数据，在redis数据库中，哈希类型的数据包括三个部分，key ：键，field：字段，value：值，一个键对应多个字段，一个字段对应一个值，字段不能重复。
	- **HSET** key field value   ：将哈希表key中的字段field的值设为value
	- **HGET** key field            ：获取储存在哈希表key中指定字段field的值
	- **HDEL** key field            ：删除储存在哈希表key中的字段field
	- **HKEYS** key                  ：获取哈希表中所有的字段
	- **HVALS** key                  ：获取哈希表中所有的值
	- **HGETALL** key              ：获取哈希表中指定key的所有字段和值
- 列表常用操作命令：
	- 简述：类似于栈，后进先出，
	- **LPUSH** key value1 [value2]    ：将一个或多个值插入列表头部
	- **LRANGE** key start stop          ：获取列表指定范围内的元素，0，-1代表获取全部
	- **RPOP** key                               ：移除并获取列表最后一个元素
	- **LLEN** key                                ：获取列表长度
	- **BRPOP** key1 [key2] timeout  ：移除并获取列表最后一个元素，如果没有元素会阻塞列                                                     表直到等待超时或发现可弹出元素为止
- 集合常用命令：
	- 简述：集合是无序且不重复的
	- **SADD** key member1 [member2]  ： 向集合中添加一个或多个成员
	- **SMEMBERS** key                            ： 返回集合中的所有成员
	- **SCARD** key                                    ： 获取集合的成员数
	- **SINTER** key1 [key2]                       ： 返回给定所有集合的交集
	- **SUNION** key1 [key2]                     ： 返回给定所有集合的并集
	- **SDIFF** key1 [key2]                          ：返回给定所有集合的差集
	- **SREM** key member1 [member2]   ：移除集合中一个或多个成员
- 有序集合常用命令：
	- 简述：有序集合不允许存在重复的元素，每个元素都关联一个double类型的分数                       （score），redis通过分数来为集合中的成员进行从小到大排列，分数可以重复
	- **ZADD** key score1 member1 [score2 member2]   ：向有序集合中添加一个或多个成                                                                                        员，或更新成员分数
	- **ZRANGE** key start stop [WITHSCORES]                ：通过索引区间返回有序集合中指定                                                                                    区间的成员
	- **ZINCRBY** key increment member                         ：为成员的分数增加increment
	- **ZREM** key member [member]                               ：移除有序集合的一个或多个成员
- 通用命令：
	- **KEYS** pattern  ：查找所有符合给定模式pattern的key
	- **EXISTS** key     ：检查给定的key是否存在
	- **TYPE** key        ：返回key所储存的值的类型
	- **TTL** key           ：返回key剩余的存活时间
	- **DEL** key           ：当key存在时，删除key
