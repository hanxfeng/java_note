
- # 业务场景

	- 用户点击获取验证码
	
	- 后端生成随机数字验证码
	
	- 将手机号作为 key 验证码作为 value 存入 Redis，设置过期时间
	
	- 用户提交验证码时，从 Redis 中取出数据进行对比
	
	- 验证通过后删除验证码


- # 获取验证码示例

```java
@PostMapping("sendMsg")
public Result sendMsh(@RequestBody User user) {
	// 获取手机号
	String phone = user.getPhone();
	
	// 判断手机号是否为空
	if(!StringUtils.isEmpty(phone)) {
		// 生成验证码
		String code = ValidateCodeUtils.generateValidateCode(4).toString();
		
		// 没有阿里云短信服务，使用log输出  
		log.info("code = {}",code);
		
		// 将生成的验证码存放至 Redis
		redisTemplate.opsForValue().set(phone,code,5,TimeUnit.MINUTES);
		
		return Result.success("验证码发送成功");
	}
	return Result.error("验证码发送失败");
}
```


- # 登录校验示例

```java
@PostMapping("/login")  
public Result<User> login(@RequestBody Map user, HttpSession session) {
	// 获取手机号
	String phone = user.get("phone").toString();
	
	// 获取验证码
	String code = user.get("code").toString();
	
	// 获取 Redis 中的验证码
	String redisCode = redisForTemplate.opsValue().get(phone);
	
	// 进行验证码对比
	if (code != null && redisCode.equals(code)) {
		// 忽略新用户则注册部分
		// 登录成功则删除 Redis 中的验证码
		redisTemplate.delete(phone);
		
		return Result.success("登陆成功");
	}
	
	return Result.error("验证码错误")
}
```
