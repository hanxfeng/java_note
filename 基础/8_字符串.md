- 概述：
	- String 是 java 定义好的一个类，在 java.lang 中，但使用时不需要导入，所有字符串对象都是此类的对象
	- 字符串的值在创建后不可更改
	- 对字符串的修改会创建一个新的字符串
- 创建方式：
	- 1.直接赋值：`String 变量名 = "值";`
	- 2.通过新建对象的方法来创建
		- 使用空参构造创建一个空字符串：`public String();`
- 字符串的比较：
	- 不忽略大小写的比较：`a.equals(b);`
	- 忽略大小写的比较：`a.equalsIgnoreCase(b);`
- StringBuilder
	- 概述：StringBuilder 可以看成一个容器，创建后里面的内容是可变的，用于提高字符串操作效率
	- 构造方法：
		- 空参构造：`StringBuilder sd = new StringBuilder();` 创建一个空的可变字符串对象，不含有任何内容
		- 含参构造：`StringBuilder sd = new StringBuilder(String str);` 创建一个值为 str 的字符串对象
	- 常用方法：
		- `.append(任意类型);`  添加数据，返回对象本身
		- `.reverse();`  反转容器中的内容
		- `.length();` 返回字符串长度
		- `.toString();`  将 StringBuilder 转为 String
- StringJoiner：
	- 概述：StringJoiner 与 StringBuilder 类似，也是创建一个容器进行字符串操作，主要进行复杂的字符串拼接操作
	- 构造方法：
		- `StringJoiner sj = new StringJoiner("间隔符号");`  创建一个 StringJoiner 对象，指定拼接时的间隔符号
		- `StringJoiner sj = new StringJoiner("间隔符号", "开始符号", "结束符号");`  创建一个 StringJoiner 对象，指定拼接时的间隔符号，开始符号和结束符号
	- 常用方法：
		- `.append(任意类型);`  添加数据，返回对象本身
		- `.length();` 返回字符串长度
		- `.toString();`  将 StringBuilder 转为 String
