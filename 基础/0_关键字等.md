
- # private：

	- 被 private 修饰的成员（变量或方法）只能在本类中使用


- # static：

	- 表示静态，是 java 中的修饰符，可以修饰成员变量和成员方法
	
	- ## 修饰成员变量：
	
		- 当成员变量被 static 修饰时，表示该变量被该类的所有对象共享，可以使用 `类名.变量名` 或 `对象名.变量名` 进行调用，一般使用类名进行调用


- # this 的内存原理

	- ## this 作用：
	
		- 区分成员变量和局部变量，this 引用的是成员变量
	
	- ## this 本质：
	
		- 代表方法调用者的地址值
	
	- ## this 举例理解：
	
		- 使用 new 新建方法 s ，方法 s 中有函数 `setA(a)`：`this.a = a` 那么，当调用`s.setA(3)` 时，this 代表方法调用者 s 的地址值，这里假设是 001 ，a 是离它最近的变量，这里是传入的形参 3 ，这段代码在内存中就是将 3 赋值给地址 001 即 s 的 a。
	
	- ## 说明：
	
		- 在Java中，非静态方法通过隐含的this参数知道自己是哪个对象的方法，所以能访问那个对象的属性和方法。静态方法没有this，它不知道自己属于哪个对象，因此不能直接访问需要特定对象才能存在的非静态成员。


- # main 方法的解释：

	- `String [] args` 是早先版本用于获取键盘输入的，现在不使用，不过仍被保留下来


- # super：

	- 用于访问父类，代表父类储存空间


- # final ：

	- 根据修饰的是类/方法/变量，作用有所不同，但大体都是不能再被改变的意思
	
	- ## 类：
	
		- 当 final 修饰类时，代表这个类是最终类，不能再被继承
	
	- ## 方法：
	
		- 当 final 修饰方法时，代表这个方法是最终方法，不能再被重写
	
	- ## 变量：
	
		- 修饰变量时代表这个变量是常量不能再被修改
		
		- **基本数据类型**：
		
			- 变量值不能再修改
		
		- **引用数据类型**：
		
			- 地址值不能修改，但内部的值可以再修改
