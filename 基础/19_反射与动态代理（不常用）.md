- # 概述：

	- 反射允许对封装类的字段、方法和构造函数的信息进行编程访问
	  
	- java 中 Constructor 表示构造方法，Field 表示成员变量，Method 表示成员方法 


- # 获取 class 对象：

	1. `Class.forName("全类名")`
	
		- `Class c = Class.forName("全类名");`
		  
		- 最为常用
	
	2. `类名.class`
	
		- `Class c = 类名.class;`
		
		- 更多是作为参数进行传递
	
	3. `对象.getClass();`
	
		- `Class c = 对象.getClass();`
		
		- 当已经有了这个类的对象时才可以使用

- # 动态代理：

	- ## 概述：
	
		- 无侵入式的给代码增加额外功能（不修改原有代码增加功能）
		  
		- 对象有什么方法要代理，代理中一定要有对应方法
		  
		- 需要先写一个方法和代理都实现的公共接口，然后方法和代理都要实现这个接口，接口中的方法是要进行代理的方法
	
	- 代码实现：
	
		- java 中有一个 `java.lang.reflect.Proxy` 类提供为对象产生代理对象的方法
		  
		- ![[代理实现.png]]
		  
		- 返回值就是代理
```java
		// Star 是一个接口，里面的方法是需要被代理的方法
		// Bigstar 是要被代理的方法
		public static Star createProxy (Bigstar bigStar){
			Star star = (Star) Proxy.newProxyInstance(
				ProxyUtil.class.getClassLoader,
				new Class[]{Star.class},
				new InvocationHandler(){
				@Override
				public Object invoke(Object proxy,Method method,Object[] args) throws Throwable {
					// 参数一：代理的对象
					// 参数二：要运行的方法，这里的例子是 sing 或 dance
					// 参数三：调用 sing 方法时传递的实参
					具体要添加的功能的实现代码
					return method.invoke(bigstar,args) 
					// 这里相当于返回 invoke 方法的运行结果
					}
				
				}
				
			);
			return Start
		}
		// 第一个参数：大部分情况下可以作为固定写法
		// 第二个参数：传入的是要继承的接口的字节码对象
		// 第三个参数：传入的是一个匿名内部类，拦截并处理代理对象上的所有方法调用,这个匿名内
		// 部类定义了代理对象“被调用时该做什么”，是动态代理增强功能的核心所在
```
```java
		// 调用示例
		BigStar bigStar = new BigStar("坤哥");
		Star proxy = ProxyUtil.createProxy(bigStar);
		String result = proxy.sing("及你太美");
```
