- 概述：
	- 线程：操作系统能够进行运算调度的最小单位，被包含在进程中，是进程中的实际运行单位
	- 进程：程序的基本执行实体
	- 并发：同一时刻，有多个指令在单个 cpu 上交替执行
	- 并行：同一时刻，有多个指令在多个 cpu 上同时执行
- 实现方式：
	1. 继承 Thread 类的方式实现
		- 在书写类时继承 Thread 类，然后在类中重写 run 方法，调用时使用 `.start()` 进行调用
		- ![[多线程第一种实现方法.png]]
	2. 实现 Runnable 接口的方式实现
		- 先实现 Runnable 接口，然后在类中重写 run 方法，创建 Runnable 的对象后创建Thread 对象，在创建时将 Runnable 对象传入，再调用`.start()` 方法
		- ![[多线程实现第二种方法.png]]
