
- # 方法的定义：
	
	`public static 返回值类型 方法名 (参数1，参数2){`
	`语句`
	`return 返回值;`
	`}`


- # 方法的重载：
	
	在同一个class类中，可以定义多个同名的方法，具有类似功能，每个方法参数类型和数量不同，这些同名的方法就构成了重载关系。返回值类型可以不同，例：
```
	public class AddNumber {
		public int addNumber (int a, int b) {
			return a + b;
		};
		
		public Long addNumber (Long a, Long b) {
			return a + b;
		};
	};
```
