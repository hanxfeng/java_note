- 数组的静态初始化：
	- 静态初始化指在创建的同时进行赋值，而不是创建空数组再向里面添加数据

	- 静态初始化的数组长度是固定的
	
	- 数组也存在数据类型，但向里面存数据时java会自动进行隐式转换
	
	- 完整静态初始化格式：
		- `数据类型 [ ] 数组名 = new 数据类型 [ ] {值1,值2};`
		- `int [] arr = new int [] {1,2,3};`
		
	- 简化格式：
		- `数据类型 [ ] 数组名 =  {值1,值2};` 
		- `int [] arr = {1,2,3};`
		
- 数组中元素的访问：
	- java中数组索引从0开始
	- 获取：
		- `数组名[索引]`
		- `arr[5]`
	- 修改：
		- `数组名[索引] = 值`
		- `arr[5] = 3`
		
- 数组的遍历：
	`for (i=0; i < arr.length; i++){`
		`语句;`
	`}`
	
	- arr.length可以获取数组长度
	
- 动态初始化：
	- 动态初始化指在创建数组时只指定长度，由java自动赋初始值，不同数据类型的初始值如下：
		整数：0
		小数：0.0
		布尔值：false
		字符：`'\u0000'`  意为空字符
		字符串：null
		
	- 创建方式：
		- 数组类型 [] 数组名 = new 数据类型 [数组长度];
			`int [] arr = new int [3];`
		
内存原理：
		有`int[] arr1 = {1, 2, 3}; int[] arr2 = arr1;`arr1和arr2指向的是同一个数组，修改一个也会影响另一个
