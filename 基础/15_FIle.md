- 概述：File 对象表示一个路径，可以是文件路径，也可以是文件夹路径，也可以是不存在的路径
- 创建：
	- 根据文件路径创建文件对象：
		- `File f = new File(路径);`
	- 根据父路径名字符串和子路径名字符串创建文件对象：
		- `File f = new File(父路径,子路径);`
			- 父路径指的是当前路径的上一级目录，例如，路径 `/home/user/documents/file.txt` 的父路径是 `/home/user/documents`
			- 子路径是相对于某个路径的相对路径，例如，在路径 `/home/user` 下，`documents/file.txt` 就是一个子路径，子路径可以是多级的
	- 根据父路径对应 file 对象和子路径名字符串创建文件对象
		- `File f = new File(父文件对象,子路径);`‘
